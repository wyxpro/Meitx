# PPT 模板说明

所有模板存储在云端，使用前通过下载脚本下载到 `/tmp/` 目录：

```bash
python scripts/download_template.py 风格名
# 示例：
python scripts/download_template.py 干净商务风
```

模板下载地址由 `scripts/templates.json` 维护，**不要手动拼接或修改 URL**。

---

## 模板一览

### 干净商务风
- **幻灯片数**：40 页
- **风格**：简洁专业，深色/浅色分区，配色以黑白灰为主，辅以单一强调色
- **包含页型**：封面、目录、章节分隔、人员介绍、服务展示、数据图表（条形/饼图/进度条）、SWOT 分析、时间轴、社交媒体数据、地图页、致谢页
- **适用场景**：企业介绍、品牌推介、年度报告、商业提案、投资路演、战略规划、数据密集型汇报

---

### 文艺风格
- **幻灯片数**：23 页
- **风格**：暖色系，手工感强，配色柔和（米色/橙棕/深绿），插图与装饰元素丰富
- **包含页型**：封面、章节分隔、议程/日程、团队介绍、项目展示、数字标注、关于我们、致谢
- **适用场景**：创意行业（设计、摄影、艺术）展示、品牌故事、文化推广、节日活动策划、个人作品集

---

### 卡通几何风
- **幻灯片数**：20 页
- **风格**：扁平几何图形，配色鲜艳活泼（蓝/橙/绿），中文友好
- **包含页型**：封面（中文标题）、目录、章节分隔、内容页（文字+图标）、数据图表、致谢
- **适用场景**：工作总结汇报、季度/年度回顾、团队分享、内部培训、学校课件

---

### 绿色植物系
- **风格**：清新自然，以绿色为主色调，配合植物插图和柔和背景
- **适用场景**：环保、可持续发展、农业、健康类主题；自然科普、生态报告、CSR 报告；养生健康类内容

---

### 赛博朋克风
- **风格**：暗色背景，霓虹配色（紫/青/粉），强烈科技感和未来感
- **适用场景**：科技产品发布、AI/区块链/元宇宙、游戏行业、电竞活动、需要强视觉冲击感的创意展示

---

### 古风水墨画风格
- **风格**：水墨画元素，留白构图，古典中式配色（黑/灰/赭/金）
- **适用场景**：中国传统文化、历史、诗词、国学；博物馆、文旅、文化品牌推介；需要体现东方美学的场合

---

## 模板选择决策

| 用户需求关键词 | 风格名 |
|---|---|
| 商务、企业、提案、路演、数据、图表、SWOT、时间轴 | 干净商务风 |
| 创意、艺术、摄影、设计、活动、文化、节日 | 文艺风格 |
| 工作总结、年度汇报、培训、课件、学校 | 卡通几何风 |
| 自然、环保、健康、清新、农业、生态 | 绿色植物系 |
| 科技、游戏、AI、元宇宙、未来感、暗黑 | 赛博朋克风 |
| 中国传统、国风、水墨、诗词、文旅、历史 | 古风水墨画风格 |

需求模糊时，根据主题内容、受众、语气综合判断最合适的风格。

## 页数增删规则

模板页数不等于最终输出页数，可根据内容需要自由增删：

- **删除幻灯片**：从 `ppt/presentation.xml` 的 `<p:sldIdLst>` 中移除对应条目，并删除 `ppt/slides/slideN.xml` 及其 `_rels` 文件。具体操作见 [editing.md](../references/editing.md)。
- **复制/新增幻灯片**：使用 `python scripts/add_slide.py ./unpacked/ slideN.xml` 复制已有版式，再修改内容。
- **不要为了凑页数保留内容为空的幻灯片**，宁少勿滥。
